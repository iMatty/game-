About

"Game$" is a Single Page Application made for people to find out if game is avilable to buy. Our website offers four different ways to login. Thanks to that we can add games to observe panel and see actual price or discount. We can also group search data via price, name, type. At this time we can offer to find games at one time from two platforms Steam and GOG. As a developers we can assure that this project is going to be continued and become even greater.
